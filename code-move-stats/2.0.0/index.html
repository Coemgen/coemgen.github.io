InitCodeMoveTemplate.js

This script initializes the Code Move Counts Template Spreadsheet.

Before running the script, values must be set for the following script properties:

  • codeMoveTemplateId – the spreadsheet id for the montly totals template
  • groupEmail – the Google Group email associated with this project

Run the script using the V8 Runtime.

MonthlyRun.js

This script creates the file structure used for compiling and viewing stats. Create a script trigger to run its driver, MonthlyRun.main(), once a month.

Before running the script, values must be set for these script properties:

  • yearlyStatsTemplateId – the spreadsheet id for the yearly stats template
  • dataFolderId – the parent folder for yearly data folders
  • codeMoveTemplateId – the spreadsheet id for the montly totals template

Run the script using the V8 Runtime.
